# PyCrypt

## Description
A comprehensive utility that allows encryption and decryption through multiple cryptographic algorithms.

## Features
- **Encryption**: Secure data encoding using standard algorithms
- **Decryption**: Recover original data from encrypted content
- **Multiple Algorithms**: Support for various encryption methods
- **User-Friendly**: Simple interface for easy operation
- **Force Decryption**: Attempt decryption on various cipher keys and display most plausible results

## Requirements
- Python 3.7+
